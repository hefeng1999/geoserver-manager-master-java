## Using the library 

### Working with Maven 

```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>


 <dependency>
    <groupId>com.github.hefeng1999</groupId>
      <artifactId>geoserver-manager-master-java</artifactId>
    <version>91e9ffed78</version>
</dependency>
```
